# documentation

$networking¶

$networking will solve your problem of where to start with documentation, by providing a basic explanation of how to do it easily.

Look how easy it is to use:

import project # Get your stuff done project.do_stuff()

Basic Tasks¶

  • disable ipv6:

    $ sudo sysctl -w net.ipv6.conf.eth0.disable_ipv6=1

-add to /etc/sysctl.conf:

$ net.ipv6.conf.eno0.disable_ipv6=1

Firewall¶

  • firewall to allow dns qrys

    $ sudo firewall-cmd –zone= –add-service=dns
